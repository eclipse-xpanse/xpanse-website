Credentials Management

APIs for managing user's cloud provider credentials

updateCredential

Update user's credential for connecting to the cloud service provider.
Required role: admin or user

Authorizations:
OAuth2 Flow
Request Body schema: application/json
name
required
string

The name of the credential

xpanseUser
required
string

The user who create the credential.

csp
required
string
Enum: "huawei" "flexibleEngine" "openstack" "alicloud" "aws" "azure" "google"

The cloud service provider of the credential.

description
string

The description of the credential

type
required
string
Enum: "variables" "http_authentication" "api_key" "oauth2"

The type of the credential

required
Array of objects (CredentialVariable)

The variables list of the credential

timeToLive
required
integer <int32>
Default: 3600

The time in seconds to live of the credential

Responses

Generated server url

http://localhost:8080/xpanse/auth/csp/credential

Monitor

APIs to get metrics of deployed services.

getMetricsByServiceId

Get metrics of all resources in a deployed service.
Required role: admin or user

Authorizations:
OAuth2 Flow
path Parameters
id
required
string

Id of the deployed service

query Parameters
monitorResourceType
string
Enum: "cpu" "mem" "vm_network_incoming" "vm_network_outgoing"

Types of the monitor resource.

from
integer <int64> >= 0

Start UNIX timestamp in milliseconds. If no value filled,the default value is the UNIX timestamp in milliseconds of the five minutes ago.

to
integer <int64> >= 0

End UNIX timestamp in milliseconds. If no value filled,the default value is the UNIX timestamp in milliseconds of the current time.

granularity
integer <int32>

Return metrics collected in provided time interval. This depends on how the source systems have generated/collected metrics.

onlyLastKnownMetric
boolean
Default: false

Returns only the last known metric. When this parameter is set then all other query parameters are ignored.

Responses

Generated server url
