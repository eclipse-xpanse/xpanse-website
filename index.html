Get Started

Portable Services

An Xpanse Cloud service is portable. It can be deployed on any cloud provider hosting Xpanse runtime.

Managed Services

Creating a service with Xpanse is very easy. You describe a service and its deployment details using the Xpanse service descriptor.

Services Catalog

As a service is described using Xpanse service descriptor, it's pretty easy to provide a catalog of services.

API First

Xpanse provides REST API for registering new services to the catalog and also for ordering services from the catalog.

Deployment Tools

Xpanse uses standard deployment tools such as Terraform for deploying services. Scripts are provided as part of the Xpanse service descriptor.

No Lock-In

Xpanse allows services and data interoperability within cloud providers and service portability across multiple cloud vendors.
